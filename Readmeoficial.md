# Comarca Ventas - Official README

This repository contains the Comarca Ventas project: a local marketplace mobile app.

## Quickstart
See docker-compose.yml for local development. Use Expo for mobile frontend.

## Project layout
- backend/: NestJS backend
- app/: React Native (Expo) app
- docs/: documentation and wireframes
- terraform/: infra starters

## Contribution
Fork -> feature branch -> PR -> review -> merge to main.
