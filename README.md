# Comarca_ventas
La aplicación móvil diseñada para potenciar la eficiencia de tu equipo de ventas. Ideal para empresas con vendedores en ruta, esta app simplifica y automatiza las tareas diarias, permitiendo a tu personal enfocarse. Optimiza tu proceso de ventas y gestión de clientes. Gestiona pedidos, inventarios y cobranzas desde tu dispositivo móvil.
